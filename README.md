# CityTwin – AI Powered City Hazard Precaution System

## Overview
CityTwin predicts urban hazards such as floods using AI models and
simulation-driven risk analysis.

## Project Structure
- mini_project/CityTwin/backend – AI models and simulations
- mini_project/CityTwin/frontend – UI layer
- Dockerfile – containerized deployment

## How to Run
```bash
pip install -r mini_project/CityTwin/backend/requirements.txt
python mini_project/CityTwin/backend/main.py
